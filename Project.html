PercIV Portfolio - Project

Church Scheduler Application

A Collab project where you can reserve a mass schedule for a church and it is saved in the database. It is created with Visual Studio C# and SQLite for the database.

Organization Profile

A C# Form Application that checks the input of the user on the form whether it is legitimate or not. When the input is validated then is shows a new form of the compilation of the data into an information about the student.

Math Quiz Website

A HTML, CSS, and JavaScript website aimed as a quiz application about Math problems that comes with Audio Sound Effects and Music. This comes with three difficulties: Easy, Medium, and Hard about Algebra. This application was made during the first year in Senior High School.

Exprint Layout

A map layout of a printing shop created in Source Engine with Half-Life 2. It is documented with the connections of the network of devices that are placed around the shop.

Minefield Game

A C++ console application game. This is like minesweeper but it is one directional and you have to keep in mind about the coordinates of the noted mines and use decision making whether you are going to step on a tile or mark it if you're sure it has a mine.
